LoudVoice replaces the basic WordPress comments by a <strong>powerful comment system</strong> that includes logging in with 30+ social networks,
spam filters and more.
 
<strong>Features</strong>
<ul>
 <li>Post as guest or login with 30+ social networks</li>
 <li>Comments are indexable by search engines and optimized for SEO</li> 
 <li>Existing comments can be imported</li>
 <li>New comments are automatically synchronized with your WordPress database</li>
 <li>Powerful moderation and admin tools</li>
 <li>Full spam filtering, blacklists and whitelists</li>
 <li>Single moderation interface for comments on multiple WordPress blogs</li>	
</ul>

== Installation ==

= Plugin Installation =
1. Upload the plugin folder to the "/wp-content/plugins/" directory of your WordPress site,
2. Activate the plugin through the 'Plugins' menu in WordPress,
3. Visit the "LoudVoice" administration page to setup the plugin. 

= API Connection =
The social network APIs are constantly changing and being updated. We monitor these changes and automatically 
update our APIs, so that you can be sure that Social Sharing Icons will always run smoothly and with the most up-to-date 
API calls. 

In order to enable the plugin you must connect with the OneAll API and create a free account at https://app.oneall.com

== Frequently Asked Questions ==

= Do I have to add template tags to my theme? =

You should not have to change your templates. 
LoudVoice seamlessly integrates into your blog by using predefined hooks.

= Do I have to change my Rewrite Settings? =

The plugins does not rely on mod_rewrite and does not need any additional rules.
It should work out of the box.
